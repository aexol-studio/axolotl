# Bookings Hotels

Provides GraphQL Schema and backend for hotel reservations

## Instalation

```sh
npx @aexol/modularium@latest add boookings-services
npm i mongodb i-graphql
```


## Environment variables

### Required
- **MONGO_URL** - url of your mongodb database
- **JWT_SECRET** - url of your mongodb database
- **MAIGUN_SENDER** - email of sender for example noreply@example.com
- **MAILGUN_DOMAIN** - Mailgun main domain for example example.com
- **MAILGUN_SERVER_DOMAIN** - Domain EU/US of mailgun service
- **MAILGUN_API_KEY** - Api key from mailgun
